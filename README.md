# YARDNEST Green-Carrot Website

A single-file GitHub Pages demo website for a high-value, low-asset-risk civic marketplace concept inspired by INBY-style backyard micro-shelter coordination.

## Files

- `index.html` — full website, styling, interactive calculator, mock paywall, buttons, tabs, and app-style demo.

## How to use on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. GitHub will publish the site after a short delay.
